# Hana El-Ashry — Website

This repo contains a simple bilingual (Arabic / English) static website for the law office "Hana El-Ashry" ready to be published to Netlify or GitHub Pages.

Deploy to Netlify (recommended):
1. Create a Netlify account at https://app.netlify.com/signup and connect your GitHub account.
2. In Netlify, select "New site from Git" → GitHub → choose the repo `hana-el-ashry-site`.
3. Build command: (leave empty). Publish directory: `/`.
4. Deploy site. Netlify will assign a `*.netlify.app` domain and provide HTTPS.

Netlify Forms:
- The contact forms include `data-netlify="true"` so Netlify will capture submissions and show them under Site → Forms. No backend required.

Replace assets/hero.jpg and assets/logo.svg with your real photos or logo.
